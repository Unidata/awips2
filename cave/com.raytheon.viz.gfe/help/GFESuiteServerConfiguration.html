GFESuite Configuration Guide



The GFESuite Configuration Guide describes the various methods to configure GFE.

General Description General description describing how GFE is configured.
Syntax for serverConfig.py Description of the serverConfig configuration file.
Syntax for localConfig.py Description of the site modification file to serverConfig.

Syntax for Maps.py Baseline map background configuration file.
Syntax for MapFiles.py Baseline map background shapefile name configuration file.
Syntax for localMapFiles.py Override file for using different (e.g., updated) shapefiles for map backgrounds.
Syntax for localMaps.py Override file for customizing map backgrounds.
Syntax for VTECPartners.py VTEC configuration for coordination and decoding.
Syntax for localVTECPartners.py Override file for the VTEC configuration.
