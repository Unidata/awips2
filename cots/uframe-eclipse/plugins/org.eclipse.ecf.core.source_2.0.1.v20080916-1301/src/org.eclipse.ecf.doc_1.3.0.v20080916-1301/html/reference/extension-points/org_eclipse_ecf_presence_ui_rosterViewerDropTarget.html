Roster Viewer Drop Target

Identifier:
org.eclipse.ecf.presence.ui.rosterViewerDropTarget

Since:
1.0.0

Description:
This extension point allows drop targets to be specified for the MultiRosterView.

Configuration Markup:

<!ELEMENT extension (dropTarget+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT dropTarget EMPTY>

<!ATTLIST dropTarget

class CDATA #REQUIRED>

Drop target element.


  • class - Class for roster viewer drop target. Must implement org.eclipse.ecf.presence.ui.dnd.IRosterViewerDropTarget. Required.

Examples: 

   <extension
         point="org.eclipse.ecf.presence.ui.rosterViewerDropTarget">
      <dropTarget
            class="org.eclipse.ecf.presence.collab.ui.TextRosterViewerDropTarget">
      </dropTarget>
   </extension>

Supplied Implementation:
No supplied implementation.
